# CC Hotel System
Uses ComputerCraft and Advanced Peripherals to make a system that checks players into a hotel.

## How to Use:
1) On your main front desk computer, run the command: wget https://raw.githubusercontent.com/OneLinkPog/cc-hotel-system/main/hotel_server.lua
2) On each room computer, run: wget https://raw.githubusercontent.com/OneLinkPog/cc-hotel-system/main/room_door.lua
